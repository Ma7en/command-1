## 🛠️ Start Migrate :-

<h3 align="center"> Migrate For Ubuntu </h3>

`Step 1` : Create File.

```
migrations
```

```
__init__.py
```

`Step 2` : Create Migrate.

```
python3 manage.py makemigrations
```

```
python3 manage.py migrate
```

`Step 3` : Run Server.

```
python3 manage.py runserver
```
